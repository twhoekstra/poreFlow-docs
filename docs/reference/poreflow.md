# `poreflow`

## Overview

This page contains the reference for the main classes found in poreFlow.

## Quick links
- [`poreflow.File`][poreflow.File]: input/output to `.fast5` and `.dat` files.
- [`poreflow.RawDataFrame`][poreflow.RawDataFrame]: Table for storing full measurements on a single channel and their current/voltage data
- [`poreflow.EventsDataFrame`][poreflow.EventsDataFrame]: Table for storing events and their properties
- [`poreflow.EventDataFrame`][poreflow.EventDataFrame]: Table for storing an event and its current data
- [`poreflow.StepsDataFrame`][poreflow.StepsDataFrame]: Table for storing steps for one or more events
